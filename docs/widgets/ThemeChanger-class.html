ThemeChanger class

A widget to change the theme.

There are three ways to change the theme:

  1. By theme: set a new theme (as a ThemeData) with ThemeChanger.of(context).theme = newTheme. See ThemeChangerState.theme

  2. By key: pass a map of themes to ThemeChanger() as themes and call ThemeChanger.of(context).themeName = key. See ThemeChangerState.themeName.

  3. By brightness: pass in a light theme, dark theme, and default brightness to ThemeChanger() and call ThemeChanger.of(context).brightness = brightness. See ThemeChangerState.brightness.
